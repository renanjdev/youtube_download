# Frontend Next.js

## Setup

1. Instale as dependências:
   `ash
   npm install
   `
2. Configure NEXT_PUBLIC_API_URL para apontar para a API (ex: http://localhost:8000).
3. Rode em modo dev:
   `ash
   npm run dev
   `

A tela principal permite enviar URLs, selecionar modo (vídeo ou áudio) e monitorar o dashboard com animações e progresso simulados.
